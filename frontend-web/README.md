# Frontend Web Starter

This is a lightweight frontend client for fast usage during exam prep.

## Run
1. Start backend API on `http://127.0.0.1:8000`
2. Open `index.html` in a browser

## Features implemented
- Login and token-based API usage
- Subject loading and question selection
- Practice session creation
- Answer submission and feedback view
- Evidence snippet rendering
